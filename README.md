# the-one-where-ai-agent-skills-meets-playwright

This repository demonstrates a workspace using an AI agent skill integration with the Playwright CLI.

## Contents

- `package.json` – defines the project dependency on `@playwright/cli`.
- `skills-lock.json` – tracks the installed skill `playwright-cli` from `microsoft/playwright-cli`.
- `.agents/skills/playwright-cli/SKILL.md` – contains the skill metadata and instructions for the Playwright CLI integration.

## Dependencies

- `@playwright/cli` `^0.1.13`

## Skill

- `playwright-cli` from GitHub source `microsoft/playwright-cli`

## Usage

1. Install dependencies:

   ```bash
   npm install
   ```

2. Use the Playwright CLI or the installed skill integration as needed in this workspace.

## Notes

This repo is configured for an AI-assisted environment and includes the Playwright skill definition under `.agents/skills/playwright-cli/`.
